### I. Подборка новостей

Новостная лента состоит из $n$ последовательно расположенных записей. Каждая запись имеет положительную характеристику $a_i$ — величину, на которую улучшается настроение после её прочтения.

Лента является циклической: после последней записи снова идёт первая.

Можно начать чтение с любой записи и прочитать подряд не более $n$ записей.

Требуется найти минимальное количество записей, сумма позитивностей которых не меньше $p$:

```math
a_k+a_{k+1}+\ldots+a_{k+c-1} \geq p
```

Индексы рассматриваются по циклу.

Ограничения:

```math
1 \leq n \leq 1000
```

```math
1 \leq p \leq 10^7
```

```math
1 \leq a_i \leq 10^4
```

**Ввод:** в первой строке числа $n$ и $p$. Во второй строке записаны позитивности $a_1,a_2,\ldots,a_n$.

**Вывод:** номер начальной записи $k$ и количество записей $c$. Сначала минимизируется $c$, а при равенстве — $k$. Если решения нет, вывести `-1`.

### K. Лыжные гонки

В соревновании участвуют $N$ лыжников. Они стартуют с интервалом в одну минуту.

Лыжник с номером $i$ проходит один километр за $w_i$ минут. Длина трассы равна $L$ километрам.

Время старта лыжника с номером $i$:

```math
i-1
```

Время его финиша:

```math
t_i=(i-1)+Lw_i
```

Считается, что лыжник $i$ обогнал лыжника $j$, если он стартовал позже, но финишировал раньше:

```math
i>j
```

```math
t_i<t_j
```

Требуется определить общее количество таких пар.

Ограничения:

```math
1 \leq N \leq 5\cdot 10^5
```

```math
1 \leq L \leq 10^9
```

```math
1 \leq w_i \leq 10^9
```

**Ввод:** в первой строке числа $N$ и $L$. Во второй строке записаны значения $w_1,w_2,\ldots,w_N$.

**Вывод:** общее количество обгонов.

[Код решения](solutions/solution_K.py)
