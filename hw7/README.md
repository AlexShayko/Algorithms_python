### A. Количество маршрутов в прямоугольной таблице

Дана прямоугольная таблица размером $N \times M$.

Изначально игрок находится в левой верхней клетке. За один ход он может перейти в соседнюю клетку вправо или вниз.

Требуется определить количество различных маршрутов из левой верхней клетки в правую нижнюю.

Ограничения:

```math
1 \leq N,M \leq 10
```

**Ввод:** два натуральных числа $N$ и $M$ — размеры таблицы.

**Вывод:** количество возможных маршрутов.

### B. Ход конём

Дана прямоугольная шахматная доска размером $N \times M$.

Конь начинает движение из левого верхнего угла и может выполнять только два вида ходов:

* на две клетки вниз и одну клетку вправо;
* на одну клетку вниз и две клетки вправо.

Требуется определить количество различных маршрутов в правый нижний угол доски.

Ограничения:

```math
1 \leq N,M \leq 50
```

**Ввод:** два натуральных числа $N$ и $M$.

**Вывод:** количество способов добраться до правого нижнего угла.

### C. Треугольник Паскаля

Треугольник Паскаля начинается со строки, содержащей одно число $1$.

Первый и последний элементы каждой строки равны $1$. Каждый внутренний элемент равен сумме двух элементов предыдущей строки:

```math
C_n^k=C_{n-1}^{k-1}+C_{n-1}^{k}
```

Ограничение:

```math
0 \leq N \leq 30
```

**Ввод:** одно целое число $N$.

**Вывод:** первые $N$ строк треугольника Паскаля. Числа в каждой строке выводятся через пробел.

### D. Самый дешёвый путь

В каждой клетке прямоугольной таблицы размером $N \times M$ записана стоимость её прохождения.

Игрок начинает в левой верхней клетке и может двигаться только вправо или вниз. Стоимость начальной и конечной клеток также учитывается.

Требуется найти минимальную суммарную стоимость пути до правой нижней клетки.

Ограничения:

```math
1 \leq N,M \leq 20
```

Стоимость каждой клетки:

```math
0 \leq a_{i,j} \leq 100
```

**Ввод:** числа $N$ и $M$, затем $N$ строк по $M$ чисел — стоимости клеток.

**Вывод:** минимальная суммарная стоимость пути.

### E. Шашку — в дамки

На шахматной доске размером $8 \times 8$ находится одна белая шашка.

За один ход шашка может перейти на одну клетку:

* вверх и вправо;
* вверх и влево.

Шашка становится дамкой, когда достигает верхней горизонтали доски.

Требуется определить количество различных путей, которыми шашка может пройти в дамки.

**Ввод:** два числа от $1$ до $8$ — номер столбца слева и номер строки снизу, в которой находится шашка.

**Вывод:** количество возможных путей в дамки.

### F. Куча камней

Даны $n$ камней с весами

```math
w_1,w_2,\ldots,w_n
```

Требуется распределить их между двумя кучами так, чтобы модуль разности суммарных весов куч был минимальным.

Если сумма весов камней в первой куче равна $S_1$, а во второй — $S_2$, необходимо минимизировать

```math
|S_1-S_2|
```

Ограничения:

```math
1 \leq n \leq 20
```

```math
1 \leq w_i \leq 100000
```

**Ввод:** число $n$ и веса всех камней.

**Вывод:** минимальная возможная разность весов двух куч.

### G. Рюкзак

Дан рюкзак вместимостью $S$ и $N$ золотых слитков с заданными весами.

Каждый слиток можно взять не более одного раза.

Требуется выбрать слитки так, чтобы их суммарный вес не превышал вместимость рюкзака и был максимально возможным.

Ограничения:

```math
1 \leq S \leq 10000
```

```math
1 \leq N \leq 300
```

Вес каждого слитка не превышает $100000$.

**Ввод:** в первой строке числа $S$ и $N$. Далее записаны веса $N$ слитков.

**Вывод:** максимальный суммарный вес золота, который можно унести.

### I. Рюкзак с ценой предметов

Дано $N$ предметов. Для каждого предмета известны его масса $m_i$ и стоимость $c_i$.

Рюкзак выдерживает суммарный вес не более $M$.

Требуется выбрать предметы так, чтобы их суммарная стоимость была максимальной, а суммарная масса не превышала вместимость рюкзака:

```math
\sum m_i \leq M
```

Ограничения:

```math
N \leq 100
```

```math
M \leq 10000
```

```math
m_i,c_i \leq 100
```

**Ввод:** в первой строке числа $N$ и $M$. Во второй строке записаны массы предметов, в третьей — их стоимости.

**Вывод:** максимальная возможная стоимость предметов в рюкзаке.
