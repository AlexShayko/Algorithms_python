### A. Сложность двоичного поиска

Загадано целое число от $1$ до $N$.

За один вопрос можно попытаться угадать число и получить один из трёх ответов:

* число угадано;
* загаданное число больше;
* загаданное число меньше.

Требуется определить минимальное количество вопросов, которого гарантированно хватит для нахождения числа методом деления пополам.

Ограничение:

```math
N \leq 10^5
```

**Ввод:** одно натуральное число $N$.

**Вывод:** минимальное количество вопросов.

### B. Корень кубического уравнения

Дано кубическое уравнение

```math
ax^3+bx^2+cx+d=0,
```

где

```math
a \neq 0.
```

Известно, что уравнение имеет ровно один действительный корень. Требуется найти его.

Ограничения:

```math
-1000 \leq a,b,c,d \leq 1000
```

**Ввод:** четыре целых числа $a$, $b$, $c$ и $d$.

**Вывод:** единственный корень уравнения с точностью не менее пяти знаков после десятичной точки.

### C. Поиск

Дан отсортированный по возрастанию массив из $N$ целых чисел и $K$ поисковых запросов.

Для каждого запроса требуется определить, содержится ли заданное число в массиве. Поиск необходимо выполнить с помощью бинарного поиска.

Ограничения:

```math
1 \leq N,K \leq 10^6
```

Для элементов обоих массивов выполняется:

```math
|a_i| \leq 10^9
```

**Ввод:** в первой строке числа $N$ и $K$. Во второй строке записаны элементы отсортированного массива. В третьей строке записаны $K$ искомых чисел.

**Вывод:** для каждого запроса вывести `YES`, если число присутствует в массиве, и `NO` в противном случае.

### F. Дипломы

Имеется $n$ одинаковых дипломов размером $w \times h$.

Требуется разместить их на квадратной доске. Дипломы нельзя поворачивать, а их внутренние области не должны пересекаться.

Необходимо найти минимально возможную длину стороны квадратной доски.

Ограничения:

```math
1 \leq w,h,n \leq 10^9
```

На квадратной доске со стороной $x$ можно разместить

```math
\left\lfloor \frac{x}{w} \right\rfloor
\cdot
\left\lfloor \frac{x}{h} \right\rfloor
```

дипломов.

**Ввод:** три целых числа $w$, $h$ и $n$.

**Вывод:** минимальная длина стороны доски.

### G. Ксерокопирование

```math

```

**Ввод:** три натуральных числа $N$, $x$ и $y$.

**Вывод:** минимальное время в секундах.

### H. Верёвочки

Имеется $N$ верёвок и $K$ домиков.

Верёвки разрешается разрезать. Требуется получить не менее $K$ верёвочек одинаковой целой длины и максимизировать эту длину.

Если выбрать длину $x$, количество получаемых верёвочек равно

```math
\sum_{i=1}^{N}
\left\lfloor \frac{a_i}{x} \right\rfloor.
```

Ограничения:

```math
1 \leq N,K \leq 10001
```

Длина каждой исходной верёвки составляет от $1$ сантиметра до $100$ километров.

**Ввод:** в первой строке числа $N$ и $K$. В следующих $N$ строках записаны длины верёвок в сантиметрах.

**Вывод:** максимальная возможная длина одной верёвочки. Если получить $K$ верёвочек невозможно, вывести `0`.

### I. Вырубка леса

В лесу растёт $X$ деревьев. Два лесоруба работают одновременно.

Дмитрий срубает по $A$ деревьев в день, но отдыхает каждый $K$-й день.

Фёдор срубает по $B$ деревьев в день, но отдыхает каждый $M$-й день.

За $d$ дней Дмитрий работает

```math
d-\left\lfloor \frac{d}{K} \right\rfloor
```

дней, а Фёдор —

```math
d-\left\lfloor \frac{d}{M} \right\rfloor
```

дней.

Требуется определить минимальное количество дней, за которое будет вырублено не менее $X$ деревьев.

Ограничения:

```math
1 \leq A,B \leq 10^9
```

```math
2 \leq K,M \leq 10^{18}
```

```math
1 \leq X \leq 10^{18}
```

**Ввод:** пять целых чисел $A$, $K$, $B$, $M$ и $X$.

**Вывод:** минимальное количество дней.

### J. Космическое поселение

Необходимо разместить $n$ одинаковых прямоугольных модулей размером $a \times b$ на поле размером $w \times h$.

Вокруг каждого модуля можно установить защитный слой целой толщины $d$. После этого размеры одного модуля будут равны

```math
(a+2d)\times(b+2d).
```

Все модули должны иметь одинаковую ориентацию и располагаться в виде регулярной сетки.

Требуется найти максимальную возможную толщину защиты $d$.

Ограничения:

```math
1 \leq n,a,b,w,h \leq 10^{18}
```

Гарантируется, что модули можно разместить без дополнительной защиты.

**Ввод:** пять целых чисел $n$, $a$, $b$, $w$ и $h$.

**Вывод:** максимальная толщина защитного слоя. Если установить защиту невозможно, вывести `0`.


```math
O(n\log n).
```

```math
1 \leq n \leq 10^5
```

```math

```

```math
0 \leq b < m
```

```math

```

**Ввод:** пять целых чисел $n$, $a_1$, $k$, $b$ и $m$.

**Вывод:** длина наибольшей строго возрастающей подпоследовательности.

### L. Коровы — в стойла!

```math

1 < K < N
```
