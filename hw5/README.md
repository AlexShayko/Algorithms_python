### A. Мячик на лесенке

На вершине лестницы из $N$ ступенек находится мячик.

За один прыжок он может спуститься на одну, две или три ступеньки. Требуется определить количество различных маршрутов, по которым мячик может добраться с вершины лестницы до земли.

Ограничение:

```math
0 < N < 31
```

**Ввод:** одно натуральное число $N$ — количество ступенек.

**Вывод:** количество возможных маршрутов.

### B. Простая последовательность

Последовательность задаётся следующими формулами:

```math
a_{2n}=a_n+a_{n-1}
```

```math
a_{2n+1}=a_n-a_{n-1}
```

Начальные значения:

```math
a_0=1,\qquad a_1=1
```

Требуется вычислить $n$-й член последовательности.

Ограничение:

```math
n \leq 1000
```

**Ввод:** натуральное число $n$.

**Вывод:** значение $a_n$.

### E. Кузнечик и лягушки

На одной линии расположены столбики с номерами от $1$ до $N$.

Кузнечик начинает на столбике с номером $1$ и может прыгать только вперёд на расстояние от $1$ до $K$ столбиков.

На некоторых столбиках находятся лягушки. Кузнечик не может приземляться на такие столбики.

Требуется определить количество способов безопасно добраться до столбика с номером $N$.

Ограничения:

```math
1 \leq N,K \leq 32
```

Количество столбиков с лягушками:

```math
0 \leq L \leq N-2
```

На первом и последнем столбиках лягушек нет.

**Ввод:** в первой строке числа $N$ и $K$. Во второй строке число $L$. В третьей строке записаны номера столбиков с лягушками.

**Вывод:** количество безопасных способов добраться до столбика $N$.

### F. Платная лестница

Лестница состоит из $N$ ступенек. Чтобы наступить на ступеньку, необходимо заплатить указанную на ней сумму.

Требуется определить минимальную стоимость пути до верхней ступеньки.

**Ввод:** в первой строке число $N$. Во второй строке записаны стоимости ступенек снизу вверх.

### G. Банкомат

Даны номиналы банкнот и сумма $S$, которую необходимо выдать.

Количество банкнот каждого номинала не ограничено. Требуется представить сумму $S$ с помощью минимального количества банкнот.

Количество номиналов:

```math
N \leq 100
```

Для каждого номинала выполняется:

```math
1 \leq x_i \leq 10^6
```

Ограничение на требуемую сумму:

```math
S \leq 10^6
```

Все номиналы различны.

**Ввод:** в первой строке число $N$. Во второй строке номиналы $x_1,x_2,\ldots,x_N$. В третьей строке сумма $S$.

**Вывод:** в первой строке минимальное количество банкнот. Во второй строке — выбранные банкноты в любом порядке. Если представить сумму невозможно, вывести `-1`.

### H. Калькулятор

Калькулятор начинает с числа $1$ и умеет выполнять три операции:

* умножить текущее число на $2$;
* умножить текущее число на $3$;
* прибавить к текущему числу $1$.

Требуется получить число $N$ за минимальное количество операций.

Ограничение:

```math
N \leq 10^6
```

**Ввод:** одно натуральное число $N$.

**Вывод:** в первой строке минимальное количество операций. Во второй строке — последовательность получаемых чисел, начиная с $1$ и заканчивая $N$.

### I. НВП

Дана последовательность из $N$ целых чисел.

Требуется найти длину её наибольшей строго возрастающей подпоследовательности.

Подпоследовательность состоит из элементов исходной последовательности, которые необязательно стоят рядом, но сохраняют первоначальный порядок.

Ограничение на длину последовательности:

```math
1 \leq N \leq 1000
```

Для каждого элемента выполняется:

```math
|a_i| \leq 10000
```

**Ввод:** в первой строке число $N$. Во второй строке записаны элементы последовательности.

**Вывод:** длина наибольшей строго возрастающей подпоследовательности.

### J. Упорядочите шарики

Имеются $N$ шариков, пронумерованных числами от $1$ до $N$. Изначально они расположены в некотором порядке.

Специальное устройство может засасывать шарики и затем возвращать их обратно. При этом устройство работает по принципу стека: последним засосанный шарик выплёвывается первым.

Засосанный шарик можно вставить в любое место последовательности. Требуется расположить шарики в порядке

```math
1,2,\ldots,N
```

и минимизировать количество операций засасывания.

Ограничение:

```math
1 \leq N \leq 1000
```

Во входной последовательности каждый номер от $1$ до $N$ встречается ровно один раз.

**Ввод:** число $N$, затем текущий порядок расположения шариков.

**Вывод:** минимальное количество операций засасывания, необходимое для упорядочивания шариков.

[Код решения](solutions/solution_J.py)
