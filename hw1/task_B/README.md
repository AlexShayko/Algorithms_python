# J. Фибоначчи по модулю [АиСД]

## Ограничения

* Ограничение времени: 1 секунда
* Ограничение памяти: 64 МБ
* Ввод: стандартный ввод или `input.txt`
* Вывод: стандартный вывод или `output.txt`

## Условие

Последовательность Фибоначчи задаётся следующим образом:

$$
F(0) = F(1) = 1
$$

Для всех $N > 1$:

$$
F(N) = F(N - 1) + F(N - 2)
$$

По заданному числу $N$ вычислите остаток от деления $F(N)$ на $10^6 + 3$.

## Формат ввода

Вводится одно целое число $N$.

$$
1 \leq N \leq 10^4
$$

## Формат вывода

Выведите $N$-е число Фибоначчи по модулю $10^6 + 3$.
